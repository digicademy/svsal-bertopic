# svsal-bertopic
Experiment for analysis of salamanca corpus with BERTopic

For now, the approach we have been working on mostly is in
[LAT_Seedwords_OpenAI.ipynb](./LAT_Seedwords_OpenAI.ipynb):

- Latin Texts
- Give Seed Words from the list of designated lemmata of the project's
  dictionary
- Use OpenAI embeddings for clustering
- Use OpenAI (gpt-4o) for topic labeling and summaries
- Use Cohere Command-R+ for additional topic labels

Analyses:

- Topic over time
- Topic by author/year
- ...
